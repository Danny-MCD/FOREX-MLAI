02

Prepare Local Environment

1. Open VS Code on your machine.

2. Create a dedicated parent folder (e.g., C:/TradingProjects/).

3. In VS Code, go to File > Open Folder and select that directory.

4. Open a new terminal within VS Code by pressing Ctrl + ` (backtick).

03

Clone Repository

Paste the following command into your VS Code terminal to download all project assets directly into your workspace:

git clone https://github.com/Danny-MCD/FOREX-MLAI.git
Pro Tip: Ensure you have Git installed on your PC before running this command.
